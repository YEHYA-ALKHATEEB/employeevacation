 When this project runs,
 the user will be ridirect to the login page
 where he can login if he is already a user
 or click on the registration link if he is not registered.


  Also the registration page includes a link to the login page.

When the user login or register he will be redirected to the home page
where he can create  a vacation.
In all this pages we have a nav-bar which include :

-If the user isn't logged in or register it will show links for register and login.

-When the user is register it will be displaying the logout button which will 
redirect the user to the login page.

Note that all this project has a responsive design using bootstrap and validations using jquery.
